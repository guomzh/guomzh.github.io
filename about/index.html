项目经验

在线问答互动平台( http://www.guomzh.com )

此平台主要功能是用户可以提出问题，其他用户回帖答复，用户可以关注用户成为粉丝，可以点赞，可以评论，还可以和关注的用户进行私信交谈。
1、用户登录模块，把用户登录状态存储在Cookie中，若用户选择记住密码，则根据客户端传来的Cookie信息，使用mysql保存的用户Token进行鉴权，实现有限期内免登录。
2、使用基于简单邮件传输协议(SMTP)的邮件发送模块(纯文本、模板邮件)完成用户注册邮箱验证
3、后端使用SpringBoot框架，前端使用模板引擎freemarker渲染整个前端模板
4、算法设计：使用自己实现的Trie树对平台发布的问题及回答进行敏感词过滤
5、用户有新动态时(如被关注了，被评论了), 使用异步队列(Redis实现)对相关用户进行站内私信或邮件通知
6、使用Redis 数据结构中的 set 集合实现用户对问题评论的点赞点踩功能。
7、solr导入mysql数据，建立问题和标题文档库，利用ik-analyzer进行中文分词，用户可以进行站内全文搜索
